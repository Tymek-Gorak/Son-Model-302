# Son302!

![Screencast_20251113_210236](https://github.com/user-attachments/assets/3310d9da-5e8b-4740-870c-4167dd7b8a42)


Grandpa's lawn is infested with pesky weeds! But his old bones have known better days, and he is unable to take care of it himself.

That's where YOU come in, as the favourite grandkid, Grandpa turns you into a lawnmower and you have no choice, but to help him out, if you want to turn back


Created for the Skansen Game Jam 2025, with the theme "Garden Keeper". The game won first place, in the local competition: https://www.skansengamejam.pl/
